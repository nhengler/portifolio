Projetos

Futstats

EM DESENVOLVIMENTO

O Futstats será meu primeiro projeto com conexão com uma API externa e criação de funcionalidades com base nela. O intuito é exibir estatísticas de determinados jogadores até o período do início da copa. Ele contará com sistema de filtros, rankings e comparações entre jogadores.


BeHealthier

Este meu primeiro projeto foi pensado para unir o útil ao agradável com duas paixões minhas: tecnologia e nutrição. Nele reúno todos os meus conhecimentos adquiridos até o presente momento. A ideia é criar um plano alimentar saudável e personalizado para o usuário.


Dashboard

O projeto consiste em um dashboard interativo que exibe dados de um banco de dados fictício de uma empresa de tecnologia. Através de gráficos e tabelas, é possível visualizar informações como salários, cargos e departamentos dos funcionários. Também é possível filtrar os dados por nível hierárquico, tipo de contrato e anos específicos. Essa experiência foi muito valiosa não só pelo fato de eu ter tido um contato mais profundo com a linguagem, mas também por ter aprendido a organizar melhor meu ambiente de desenvolvimento. Foi um aprendizado muito valioso que fortaleceu muito a forma como eu planejo meus projetos.
