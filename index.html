Projetos

BeHealthier

EM DESENVOLVIMENTO
Este meu primeiro projeto foi pensado para unir o útil ao agradável com duas paixões minhas: tecnologia e nutrição. Nele reúno todos os meus conhecimentos adquiridos até o presente momento. A ideia é criar um plano alimentar saudável e personalizado para o usuário. Atualmente, conta com apenas 2 cardápios pré-definidos, mas a intenção é fazer uma conexão com uma API externa para que os cardápios sejam gerados automáticamente de acordo com os dados e preferências do usuário.
